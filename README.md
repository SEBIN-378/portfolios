# 🌐 Personal Portfolio Website

A modern, responsive **personal portfolio website** built using HTML, CSS, and JavaScript.  
It includes authentication UI, smooth navigation, project showcase, and a contact form.

---

## 🚀 Features

✨ Login / Sign-up UI (Frontend Authentication Design)  
✨ Responsive portfolio layout  
✨ About section with personal introduction  
✨ Skills showcase grid  
✨ Projects section with live & code links  
✨ Contact form UI  
✨ Smooth navigation with anchor links  
✨ Glassmorphism-style UI design  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Glassmorphism + Responsive Design)  
- JavaScript (DOM Manipulation)
